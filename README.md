# Octopus
Octopus组件库，聚合一些有用的代码、工具类

## 聚合全文索引工具类

- 全文搜索（Lucene.NET 4.8.0-beta00016）
- Tag分类提取
- 中文分词（基于结巴，后期会考虑加入盘古）

[使用说明](Search.md)

## 谷歌云盘下载工具类

- 免调用复杂的谷歌云盘API
- 简单易用，支持大文件 ~_~

[使用说明](Google.md)

## 简单控制台进度条

- 简单好用
- 使用控制台程序进行大规模运算可以让界面显示进度，避免等待

[使用说明](ConsoleShow.md)

## AI客户端（支持OpenAI、Llama）

- 简易的AI交互客户端
- 支持对话，简易发起会话，获取AI返回的数据
- 搭配本地Llama模型，可实现批量操作，譬如翻译任何一个国家语言，自动写（新闻、小说等），自动发稿

[使用说明](AIClient.md)
